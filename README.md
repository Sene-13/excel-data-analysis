# Bike Sales Analysis & Dashboard

A comprehensive analysis of customer demographics and purchasing behavior to drive bike sales strategy.

## Project Overview

This project analyzes a dataset of 1,000 customers to identify key factors that influence bike purchases. By cleaning raw data and building an interactive dashboard in Excel, I've uncovered trends related to income, age, and commute distance to help a hypothetical bike retailer target the right audience.

## Tools Used 

Microsoft Excel: Data Cleaning, Pivot Tables, Pivot Charts, and Dashboard Design.


## Data Cleaning Process

Before analysis, I performed the following steps to ensure data integrity:

Removed Duplicates: Cleaned out 26 duplicate entries to ensure unique customer records.

Find & Replace: Standardized columns like Marital Status (changed 'M' to 'Married' and 'S' to 'Single') and Gender ('M' to 'Male', 'F' to 'Female').

Feature Engineering: Created an Age Brackets column (Adolescent, Middle Age, Old) using nested IF statements to better segment the market.

Currency Formatting: Converted the 'Income' column to a standard currency format for clarity.

## Key Insights
 
Top Buyers: Middle-aged individuals (ages 31–45) are the most frequent purchasers.

Income Factor: Customers with higher average incomes are significantly more likely to purchase a bike.

Commute Influence: People with shorter commutes (0–1 miles) have the highest conversion rate.
